Listas en HTML5

En HTML5 disponemos de tres tipos de listas que nos van a permitir organizar mejor
el texto para darle un formato más conciso y claro.
Los tipos de listas que podemos usar/disponer son los siguientes:
  • Listas Ordenadas (ol): Son listas ordenadas, van numeradas,
    que se definen como Ordered List
  • Listas Desordenadas (ul): Son listas desordenadas, no van numeradas,
    que se definen como Unordered List
  • Listas de Definición (dl): Son listas de definición, que se definen
    como Definition List

Dentro de un elemento lista, sea del tipo que sea, podemos insertar cualquier tipo
de elemento: texto, imágenes, vídeos, tablas, audios, listas, etc.
Por lo tanto, se pueden combinar listas ordenadas con desordenadas y viceversa.

Listas Ordenadas (ol)

La estructura de una lista ordenada es la siguiente:
  • La lista está formada por la etiqueta de apertura y cierre ol
  • Internamente, dispondrá de una etiqueta li en la que podremos listar
    el contenido de nuestra lista. Es decir, pondremos los elementos
    que queremos que se muestren en nuestro listado.
  • La etiqueta li irá siempre tanto en listas ordenadas como desordenadas
    porque son parte de su estructura.

Ejemplo de lista ordenada:
  1. Windows 11
  2. Ubuntu
  3. Kali Linux
  4. MacOS Yosemite
  5. Linux Mint

Listas Desordenadas (ul)

La estructura de una lista desordenada es la siguiente:
  • La lista está formada por la etiqueta de apertura y cierre ul
  • Internamente, dispondrá de una etiqueta li en la que podremos listar
    el contenido de nuestra lista. Es decir, pondremos los elementos
    que queremos que se muestren en nuestro listado.
  • La etiqueta li irá siempre tanto en listas ordenadas como desordenadas
    porque son parte de su estructura.

Ejemplo de lista desordenada:
  • Java
  • PHP
  • Python
  • JavaScript
  • C++

Formateado de Listas Ordenadas y Desordenadas

HTML5 nos permite en cierta medida poder customizar las listas a través de una serie
de atributos básicos como el atributor start y atributo type.
Con estos atributos podemos cambiar el orden de comienzo de las listas ordenadas
y el tipo de icono (podemos poner letras, números normales y romanos)
y en el caso de listas desordenadas podemos poner iconos de circulos, discos y
cuadrados.
Lista Ordenada aplicando atributos type y start
  1. Podemos poner letras en mayúscula (A)
  2. Podemos poner letras en minúsculas (a)
  3. Podemos poner cifras romanas en mayúscuas y minúsculas (i I,ii II, iii III,iv IV....n)
  4. Podemos poner números normales (1,2,3...n)
  5. Podemos indicar en qué posicion queremos que comience

Lista Desordenada aplicando atributos type
  • Podemos poner disc (viñetas redondas rellenas)
  • Podemos poner circle (viñetras redondas huecas)
  • Podemos poner square (cuadrados)

Listas de Definición (dl)

Las listas de definición tienen una estructura totalmente distinta a las anteriores.
Tienen la siguiente estructura:
  • La etiqueta dl: Description List delimita la lista completa
  • La etiqueta dt: Description Term delimitar el término
  • La etiqueta dd: Definition Description delimitar las definiciones

Ejemplo de Lista de Definición
dl
Crea la caja que contiene los items de la lista
dt
Es el Item que contiene un término a definir
dd
Es el Item que contiene una definición
